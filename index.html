Wat kan je vinden in dit museum?

In dit museum staan de codes centraal. Zowel de codes die gebruikt worden door Myrea Roskam voor het bouwen van webpagina's. Zij voegt elke keer een stukje extra kennis en informatie toe om zo uiteindelijk een volwaardige pagina te kunnen maken. In dit gedeelte van het museum is zij bezig met les 9 van HTML. Daarnaast vindt je meer soorten codes zoals ethische code, barcode, QR-code, binaire code. Elke code die je je kan bedenken, kan je vinden in dit museum. Op onze website vindt je meer informatie over het educatieve gedeelte van ons museum.
